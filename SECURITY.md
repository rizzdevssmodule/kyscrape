# Security Policy

## Supported Versions

Currently, we support the following versions of KyScrape:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of KyScrape seriously. If you find a vulnerability, please do not open a public issue. Instead, report it to the maintainers via GitHub private security advisories or email the maintainer directly.

### Our Commitment
- We will acknowledge your report within 48 hours.
- We will provide a fix or mitigation plan as soon as possible.
- We will credit you for the discovery (unless you prefer to remain anonymous).

## Transparency
KyScrape uses `playwright` to perform legitimate web automation. We do not collect user data, and we do not connect to any external command-and-control servers. All network requests are initiated by the user.
