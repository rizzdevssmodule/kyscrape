# Contributing to KyScrape

First off, thanks for taking the time to contribute!

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub issue tracker.
- Describe the bug and include steps to reproduce it.

### Suggesting Enhancements
- Explain the enhancement and why it would be useful.

### Pull Requests
1. Fork the repo.
2. Create your feature branch (`git checkout -b feature/AmazingFeature`).
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`).
4. Push to the branch (`git push origin feature/AmazingFeature`).
5. Open a Pull Request.

## Code of Conduct
Please be respectful and professional in all your interactions with the project and its maintainers.
